        # 数据引用

        变量的赋值即定义的变量中记录数据存放地址的动作叫做引用

        给已定义的变量重新赋值即变量修改为对新数据的引用

        a = 1
a = 2

        赋值语句的理解：

        1. 内存开辟一个地址存放数据

        2. 内存中定义变量 来记录数据的内存地址

        3. 访问变量即获取其记录了内存地址的数据

        如下：

        通过 id() 方法获取数据的存储地址验证

        定义的变量 a 引用数据 10 的地址，定义的变量 b 中记录变量 a 中记录的数据地址，即变量 b 也引用数据 10 ，所以 id(10) 等于 id(a) 等于 id(b)。

        将数据 20 的内存地址赋值给变量 a 之后，因为数据 20 在内存中的地址不同于数据 10，所以使得 id(a) 改变，但变量 b 还是引用数据 10 的地址所以 id(b)不变

        a = 10
print(id(10))  # 4535581264
print(id(a))  # 4535581264

b = a
print(id(b))  # 4535581264

a = 20
print(id(20))  # 4535581584
print(id(a))  # 4535581584

print(id(b))  # 4535581264

        # 局部变量

        在函数内部定义的变量

        用于在函数内临时保存该函数会用到的数据

        创建后只能在函数内部使用，函数执行结束后会被系统回收

        不同函数中的同名局部变量互不影响

        def func1():

    num = 1000
    print(num)


def func2():
  
    num = 2000
    print(num)


func1()  # 1000
func2()  # 2000

print(num) # 报错 unexpected indent

        # 全局变量

        在函数外部定义的所有函数都可使用的变量

        因为 Python 的执行是从上到下，所以全局变量需要定义在最前面

        text = 'aaa'
num = 1000


def func1():

    print(text, num)


def func2():

    print(text, num)


func1()  # aaa 1000
func2()  # aaa 1000

print(text, num)  # aaa 1000

        # global 关键字

        函数内不允许直接修改全局变量的引用

        否被视为在函数内创建了同名的局部变量

        并且函数会优先使用局部变量

        num = 100

def func():

    num = 200
    print(num)


func()  # 200
print(num)  # 100

        若要在函数内修改局部变量的引用

        需在修改前先通过 global 关键字声明一下变量

        num = 100

def func():
  
		global num
    num = 200
    print(num)


func()  # 200
print(num)  # 200

        # 不可变类型

        不可变数据类型是指内存中的数据不允许被修改

        即一旦在内存中开辟了指定位置存储了数据后就不允许被修改

        TIP

        • 数字：int，bool，float

        • 字符串：str

        • 元祖：tuple

        如下：

        通过 id() 方法获取数据的存储地址验证

        不可变类型的数据的引用修改时只能重新开辟地址，

        所以每次变量 a 修改引用时指向的数据内存地址都不同

        a = 1
print(id(a))  # 4447037744

a = 'hello'
print(id(a))  # 4460970160

a = True
print(id(a))  # 4445531760

a = False
print(id(a))  # 4445531792

        # 可变类型

        可变数据类型是指内存中的数据允许被修改

        即在内存中开辟了指定位置存储，可通过方法将开辟的内存中的内容进行修改，该数据在内存中存储的地址不会受影响

        TIP

        • 列表：list
        • 字典：dict

        如下：修改列表的内容

        通过 id() 方法获取数据的存储地址验证

        通过方法修改引用的数组只修改了内存中的数据，所以变量 a 的引用地址没有改变

        当然直接赋值语句是修改了引用地址，不是一回事

        a = [1, 2, 3]
print(id(a))  # 4460828672

a.append(4)
print(a)  # [1, 2, 3, 4]
print(id(a))  # 4460828672

a.pop()
print(a)  # [1, 2, 3]
print(id(a))  # 4460828672

a.clear()
print(a)  # []
print(id(a))  # 4460828672

a = [4,5,6] 
# 赋值语句是重新开辟内存地址并修改引用的数据地址

        # continue & break

        关键字 含义
        continue 停止当前循环，进入下一个循环
        break 结束循环，不进行后面的循环

        break 在嵌套循环中只会结束最近一层的循环，若想结束所有嵌套循环可使用exit()

        如下：continue 跳过 4

        for i in range(1, 6):
    if i == 4:
        continue
    print(i, end=' ')
