        # 2.x 与 3.x

        菜鸟 Python (opens new window)

        • Python 2.x
          • 旧版本，默认不支持中文
          • 解析器名称为 python
        • Python 3.x
          • 2008 年发布
          • 不考虑向下兼容，旧版 Python 的程序无法执行
          • 解析器名称为 python3
        • Python 2.7
          • 2010 年发布的最后的 2.x 版本
          • 考虑了 2.x 向 3.x 迁移的兼容
          • 允许使用部分 3.x 语法和函数

        无法兼容 3.x 版本时：

        先用 Python 3.x 开发，然后用 Python 2.6 或 Python 2.7 执行并做兼容处理

        仅有极少数不支持 3.x 语法的第三方库

        # % 格式化字符串

        Python 2.x 中通过%格式化字符串实现字符串中引用变量

        但比较复杂不清晰，Python 3.x 推荐 f-string 字面量格式化字符串

        格式化操作符%与不同字符组合，在字符串中起到占位符作用

        然后在字符串后通过%(参数)将相同个数的参数填充到指定占位符位置

        "xxxx%格式化符xxxx%格式化符" % (填充内容, 填充内容)

        在%和字符之间还可以使用操作符实现辅助功能

        常见格式化字符 含义
        %s 字符串
        %d 整数
        %06d 显示 6 位的整数，位数不够前面用 0 补全
        %f 浮点小数
        %.02f 显示小数后两位
        %% 一个%

        如下：

        print('价格: %.2f' %(11))  # 价格: 11.00

print('个数: %02d' %(1))  # 个数: 01

print('比例: %d%%' %(11))  # 比例: 11%

        再如下：

        name = 'Andy'
age = 28
score = 98.8

text = '''
----- %s -----
name: %.4s
age: %d
score: %.2f
----- End -----
''' % (name, name, age, score)

print(text)
'''
----- Andy Jackson -----
name: Andy
age: 28
score: 98.80
----- End -----
'''
