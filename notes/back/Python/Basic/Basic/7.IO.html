      # 基本步骤

      文件操作步骤：

      1. 打开文件，创建文件对象
      2. 读写操作
      3. 关闭文件

      Python 中通过内置函数 open() 来实现文件的输入输出操作

      # 1. 打开
file = open(文件名·路径)

# 2. 操作

# 3. 关闭
file.close

      # open()

      内置方法 open() 用于打开文件

      该方法创建并返回打开文件的对象，通过文件对象的方法来实现文本 输入输出操作

      返回一个文件流

      file = open('文件名·地址'[, mode='打开模式', encoding='字符串编写格式'])

      file_1 = open('README')
file_2 = open(r'C:\pythone\01.py')
file_3 = open('./01.py')

file_4 = open('./image.jpg', mode='rb')

      若要打开的文件不存在则报错，抛出一个 FileNotFoundError 异常

      open('./$%^*@#%.json')
# FileNotFoundError: [Errno 2] No such file or directory: './$%^*@#%.json'

      # 打开模式

      文件打开模式默在调用 open() 时指定，默认为只读模式 r

      对文本进行的操作需与打开模式一致，，否则会报错抛出一个 io 异常

      file = open('文件名·路径'[, mode='打开模式'])
      访问模式 含义
      r 只读（默认）
      w 只写，覆盖
      a 追加
      r+、w+、a+ 可读写，效率不好，不常用
      rb 读取图片文件

      文件指针：

      从哪个位置开始读写文本的标记
      打开模式为 r、r+ 时，文件指针在文本开头
      打开模式为 a、a+ 时，文件指针移到文本结尾
      频繁移动指针会影响文件读写效率，所以不建议 r+、w+、a+

      r 只读模式

      只读 r 打开的文件无法被写入内容

      a = open('01.json', 'r')
b = open('01.json')

b.write('haha')
# 报错 io.UnsupportedOperation: not writable

b.close()

      w 覆盖模式：

      若文件不存在则新建，若存在则覆盖

      file = open('./example.txt', 'w')
file.write('haha')
file.close()

      a 追加模式：

      方法执行后文件指针移动到文本末尾
      若文件不存在则新建，若存在则覆盖

      file = open('./example.txt', 'a')
file.write('haha')
file.close()

      # 编码格式

      文本文件中的字符通过字符编码实现二进制表示

      Python 2X：默认使用 ASCII 编码

      Python 3X：默认使用 UTF-8 编码

      ASCII 编码

      共有 256 个，仅包含英文字母、数字、常用转义字符
      使用一个字节表示一个 ASCII 字符，
      一个字节包含 8 个 0 或者 1 的排列组合，

      UTF-8 编码

      UTF-8 是 UNICODE 编码的一种编码格式
      几乎包含世界上所有地区的文字
      使用 1 ～ 6 个字节表示一个 UTF-8 字符（汉字使用 3 个字节）


      # 输入输出方法

      # close()

      用于操作完毕后关闭文件

      由文件对象调用

      文件操作完成后必须关闭文件，否则会造成系统资源消耗

      # 1. 打开
file = open('README')

# 2. 读写操作

# 3. 关闭
file.close

      # read()

      一次性读取返回文件的所有内容

      由文件对象调用

      一次性读取大文件会严重占用内存，更建议 readline()

      file = open('./example.txt')
content = file.read()
print(content)
file.close()

      调用 read() 之后文件指针会移动到文本内容的末尾

      所以执行过一次后便无法再读取文件内容了

      file = open('./example.txt')

a = file.read()
print(len(a))  # 文本内容长度

b = file.read()
print(len(b))  # 0

file.close()

      如下：复制小文件：

      file_from = open('example.txt')
file_target = open('example_copy.txt', 'w')

text = file_from.read()
file_target.write(text)

file_from.close()
file_target.close()

      # readline()

      一次读取一行文件内容，常用于读取大文件

      由文件对象调用

      返回读取的每一行的文本，结尾默认带一个转义字符 \n

      若读取不到则返回 None

      调用 readline() 后文件指针会移动到下一行

      file = open('./customers.txt')

while True:
    text = file.readline()

    if not text:
        break

    print(text, end="")

file.close()

      如下：复制大文件：

      file_from = open('example.txt')
file_target = open('example_copy.txt', 'w')

while True:
    text = file_from.readline()

    if not text:
      break

    file_target.write(text)


file_from.close()
file_target.close()

      # readlines()

      一次读取多行文件内容

      由文件对象调用

      返回一个列表，若读取不到则返回空列表

      file = open('./01.jpg')

for line in file.readlines():
    print(line)

file.close()

      # write()

      用于一次性将内容写入文件

      由文件对象调用

      文件对象.write('内容')

      只能写入字符串，否则报错抛出 TypeError 异常

      # TypeError: write() argument must be str

      # writelines()

      用于内容的单行写入

      由文件对象调用

      文件对象.writeline('内容')

      只能写入字符串，否则报错抛出 TypeError 异常

      # TypeError: write() argument must be str

      file = open('./01.txt', mode='w')

for line in range(4):
    file.writelines('hello\n')

file.close()

      # with 语句

      用于上下文自动管理

      with 语句结合 open() 使用

      在其作用域内书写文件操作，只要跳出作用块文件就会自动正常关闭释放资源

      省略了在每一个 open() 打开的文件流操作完后都要手写的 close()

      with open('文件名·地址') as file:
    # 输入输出操作


# 相当于简写了：
file = open('文件名·地址')
# 输入输出操作
file.close()

      with open('文件名·地址') as file_1:
    with open('文件名·地址') as file_2:
        # 输入输出操作


# 相当于简写了：
file_1 = open('文件名·地址')
file_2 = open('文件名·地址')
# 输入输出操作
file_1.close()
file_2.close()

      如下：文件复制

      将文件 01.json 内容复制到 02.json

      with open('./01.json', 'rb') as file_1:
    with open('02.json', 'wb') as file_2:
        file_2.write(file_1.read())

      原理

      with 执行时无论是否遇到异常都会自动执行 exit 内置方法，
      只要跳出 with 作用块都能保证文件正常关闭，自动实现释放资源

      class DemoClass(object):
  def __enter__(self):
    print('1.enter')
    return self

  def my_show_method(self):
    print('2.自定义方法')

  def __exit__(self,exc_type,exc_val,exc_tb):
    print('3.exit')


with DemoClass() as demo:
  demo.show()

"""
没有异常时打印结果：
1.enter
2.自定义方法
3.exit
"""

"""
有异常时打印结果：
3.exit
报错 异常类型: 异常信息
"""
