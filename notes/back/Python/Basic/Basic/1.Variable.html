# 赋值

变量的赋值实质是变量引用数据在内存中的存放地址

通过赋值后的变量可获其引用的数据

单一变量赋值

a = 1
b = 2
c = 3

print(a, b, c)
# 1 2 3

多个变量同时赋值相同数据

a = b = c = 1

print(a, b, c)
# 1 1 1

多个变量按顺序赋值不同数据

a, b, c = 1, 2, 3

print(a, b, c)
# 1 2 3

# 交换值

方法一： 临时变量

a = 1
b = 2

c = a
a = b
b = c

print(a, b)  # 2 1

方法二：元组拆包（推荐）

a = 1
b = 2

a, b = b, a
# a, b = (b, a, )

print(a, b)  # 2 1

# 局部变量

局部变量是指在函数内部定义的变量

局部变量只能在函数内部使用，函数执行结束后会被系统回收

函数内会优先使用自身局部变量，不同函数中的同名局部变量互不影响

num = 10


def func_1():
    num = 20
    return (num, id(num))


print(num, func_1()[0])  # 10 20
print(id(num) == func_1()[1])  # False



def func_2():
    num = 30
    return (num, id(num))


print(num, func_2()[0])  # 10 30
print(id(num) == func_2()[1])  # False

# locals()

查看函数内的局部变量

返回值是个字典

def func():

    print(locals())  # {}

    a = 10
    b = 10
    print(locals())  # {'a': 10, 'b': 10}

    def inner():
        return

    print(locals())  # {'a': 10, 'b': 10, 'inner': <function func.<locals>.inner at 0x109931820>}


func()

# 全局变量

全局变量是指在顶层定义的变量，需要定义在最前面

所有函数内都可获取全局变量的值

text = 'aaa'
num = 1000


def func1():
    print(text, num)


def func2():
    print(text, num)


func1()  # aaa 1000
func2()  # aaa 1000
print(text, num)  # aaa 1000

# global

出于数据安全，全局变量在函数内无法直接被修改

全局变量可在函数中通过 global 关键字声明后修改

def 函数():

    global 全局变量

    # 全局变量 = 新值
    # or 增删改全局变量

num = 10
li = [1, 2, 3]


def func_1():
    num = 20
    li = [4, 5, 6]


def func_2():
    global num, li
    num = 30
    li = [7, 8, 9]


print(num)  # 10
print(li)  # [1, 2, 3]

func_1()
print(num)  # 10
print(li)  # [1, 2, 3]

func_2()
print(num)  # 30
print(li)  # [7, 8, 9]

# 数据引用

变量和数据是分开储存在内存中

变量赋值的理解：

  1. 内存中开辟一个地址存放数据
  2. 内存中定义变量来记录数据的内存地址
  3. 访问变量即获取其记录的内存地址对应的数据

变量赋值实质赋的不是值而是值的内存地址（变量引用数据在内存中的存放地址）

给变量重新赋值即更新记录引用一个新的内存地址

a = 1
print(id(a))  # 4431444272

a = 2
print(id(a))  # 4431444304

# id()

可通过内置方法 id() 查看数据在内存中所在的地址

若参数是变量则返回其记录的数据的内存地址

id(数据)  # 数据的内存地址
id(变量)  # 变量引用的数据的内存地址

如下：

开始时变量 a、b 引用同一地址的数据

之后变量 a 被修改引用

print(id(1))  # 4431444272

a = 1
print(id(a))  # 4431444272

b = 1
print(id(b))  # 4431444272

a = 2
print(id(a))  # 4431444304

如下：

变量 a 引用了数据 hello 的储存地址，变量 b 引用了变量 a 中记录的引用地址，即变量 b、a 的引用地址相同。变量 c 也引用了数据 hello 的储存地址，所以变量 a、b、c 三者的引用地址一致

a = 'hello'
b = a
c = 'hello'

print(id(a), id(b), id(c), id('hello'))
# 4540442096 4540442096 4540442096 4540442096

# in、not in

可使用 is、is not 判断引用的数据的内存地址是否相同

详见 身份运算符

a = 'hello'
b = a
c = 'hello'

print(a is b is c)  # True

# del

回收变量，切断变量对数据地址的引用

del 变量

TIP

del 删除的是变量而不是数据

只是无法再通过该变量访问到数据，数据还存在于内存中

如下：

仅切断了变量 a 对数据 hello 的引用，回收了变量 a

hello 还存在与内存中，变量 b 依然可以引用 hello

a = 'hello'
b = a

del a

print(a)  # 报错 NameError: name 'a' is not defined
print(b)  # hello
print(id('hello') == id(b))  # True

如下： a 浅拷贝了

l = [1, 2, 3]
a = l[0]

del l[0]

print(l)  # [2, 3]
print(a)  # 1

# ❌ 深浅拷贝

结合上文 数据的引用

深浅拷贝区别：拷贝复制的是数据的内存地址，还是仅仅是数据的值

# 深拷贝

数据引用的地址相同，一个变都跟着变

a = [1, 2, 3]
d = a

print(id(a) == id(b))  # True

如下： a、b 二者的引用地址是相同的，所以 b 修改 a 也跟着变

a = [1, 2, 3]

b = a
b.append(4)

print(a)  # [1, 2, 3, 4]
print(id(a) == id(b))  # True

s = 'hello'
s_copy = s[:]
print(id(s) == id(s_copy))  # True

# 浅拷贝

在内存中开辟一个新的空间存储相同值的数据

仅数据值相同，二者存放于内存中不同的位置，互不影响

a = [1, 2, 3]

b = a.copy()
print(id(b) == id(a))  # False

c = a[:]
print(id(c )== id(a))  # False

l = [1, 2, 3]
a = l[0]

l[0] = 999
print(a)  # 1

del l[0]
print(a)  # 1

a = [1, 2, 3]

b = a.copy()
print(id(a) == id(b))  # False

b.append(4)
print(a)  # [1, 2, 3]
print(b)  # [1, 2, 3, 4]

c = a[:]
print(id(c )== id(a))  # False

c.append(4)
print(a)  # [1, 2, 3]
print(c)  # [1, 2, 3, 4]
