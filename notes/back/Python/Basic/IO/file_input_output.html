      # 基本步骤

      文件操作步骤：

      1. 打开文件，创建文件对象
      2. 读写操作
      3. 关闭文件
      file = open('README')

file.close

      # 打开模式

      file = open('文件路径', [文件打开模式])
      访问模式 含义
      r 只读（默认）
      w 只写，覆盖
      a 追加
      r+、w+、a+ 可读写，效率不好，不常用

      打开模式为 r、r+ 时，文件指针在文本开头

      打开模式为 a、a+ 时，文件指针移到文本结尾

      频繁移动指针会影响文件读写效率，所以不建议 r+、w+、a+

      ::: 文件指针：

      从哪个位置开始读写文本的标记

      :::


      # 操作函数与方法

      # open()

      用于打开文件，创建并返回要操作文件的对象

      file = open(文件地址, [打开模式, 文件中字符串编写格式])

      若要打开的文件不存在则抛出异常

      open('./$%^*@#%.json')
# FileNotFoundError: [Errno 2] No such file or directory: './$%^*@#%.json'

      文件打开模式默认为只读 r

      a = open('01.json', 'r')
b = open('01.json')

b.write('haha')
# 报错 io.UnsupportedOperation: not writable

b.close()

      # close()

      由文件对象调用，用于操作完毕后关闭文件

      文件操作完成后必须关闭文件，否则会造成系统资源消耗

      # 1. 打开
file = open('README')

# 2. 其余读写操作

# 3. 关闭
file.close

      # read()

      由文件对象调用，一次性读取返回文件的所有内容

      一次性读取大文件会严重占用内存，更建议 readline()

      file = open('./example.txt')
print(file.read())
file.close()

      调用 read() 之后文件指针会移动到文本内容的末尾

      所以执行过一次后便无法再读取文件内容了

      file = open('./example.txt')

a = file.read()
print(len(a))  # 文本内容长度

b = file.read()
print(len(b))  # 0

file.close()

      # readline()

      由文件对象调用，一次读取一行文件内容

      读取的每一行结尾自带一个转义字符 \n

      常用于读取大文件

      file = open('./customers.txt')

while True:
    text = file.readline()

    if not text:
        break

    print(text, end="")

file.close()

      调用 readline() 后文件指针会移动到下一行

      # write()

      由文件对象调用，用于将内容写入文件

      ::: 覆盖模式：

      若文件不存在则新建，若存在则覆盖

      file = open('./example.txt', 'w')
file.write('haha')
file.close()

      :::

      追加模式：

      方法执行后文件指针移动到文本末尾

      若文件不存在则新建，若存在则覆盖

      file = open('./example.txt', 'a')
file.write('haha')
file.close()

      只能写入字符串，否则报错

      # TypeError: write() argument must be str

      复制小文件：

      file_from = open('example.txt')
file_target = open('example_copy.txt', 'w')

text = file_from.read()
file_target.write(text)

file_from.close()
file_target.close()

      复制大文件：

      file_from = open('example.txt')
file_target = open('example_copy.txt', 'w')

while True:
    text = file_from.readline()

    if not text:
      break

    file_target.write(text)


file_from.close()
file_target.close()

      # 上下文管理

      可自动管理上下文

      with 执行时无论是否遇到异常，都会自动执行 exit 内置方法，

      只要跳出 with 作用块都能保证文件正常关闭，自动实现释放资源

      如下：自动实现上下文管理，不需要手动关闭文件

      class DemoClass(object):
  def __enter__(self):
    print('1.enter')
    return self

  def my_show_method(self):
    print('2.自定义方法')

  def __exit__(self,exc_type,exc_val,exc_tb):
    print('3.exit')


with DemoClass() as demo:
  demo.show()

"""
没有异常时打印结果：
1.enter
2.自定义方法
3.exit
"""

"""
有异常时打印结果：
3.exit
报错 异常类型: 异常信息
"""

      # 文本编码格式

      文本文件中的字符通过字符编码实现二进制表示

      Python 2X：默认使用 ASCII 编码

      Python 3X：默认使用 UTF-8 编码

      # ASCII 编码

      共有 256 个，仅包含英文字母、数字、常用转义字符

      使用一个字节表示一个 ASCII 字符，

      一个字节包含 8 个 0 或者 1 的排列组合，

      # UTF-8 编码

      UTF-8 是 UNICODE 编码的一种编码格式

      几乎包含世界上所有地区的文字

      使用 1 ～ 6 个字节表示一个 UTF-8 字符（汉字使用 3 个字节）
