      # 定义与使用

      Python 的类是个模版对象，里面存放公用或私用的属性与方法

      通过 class 关键字定义

      类名是大驼峰命名法（单词首字母大写没有下划线）

      # 类实例对象

      通过调用 类名() 可创建类的实例对象

      class 类名:
   pass


对象A = 类名()
对象B = 类名()

      调用类创建的对象的内存地址不同，互不影响

      直接通过 print() 打印实例对象可获得其类名和十六进制表示的内存地址

      若想获取自定义内容需通过 类内置方法 __str__

      class Person:
    pass


andy = Person()
print(andy)
# <__main__.Person object at 0x10f1821f0>

tom = Person()
print(tom)
# <__main__.Person object at 0x10f182190>

print(id(a) == id(b))  # False

      # 方法

      # self 参数

      在类的内部定义方法时，方法必须包含第一个参数 self

      self 指向调用该方法的当前类实例对象

      class 类名:

    def __init__(self, 参数):
      self.属性 = 参数

    def __私有方法(self):
      pass

    def 实例方法(self, 参数):
      pass

    def __内置方法__(self):
      pass


实例对象 = 类名(参数)
实例对象.实例方法(参数)

      class Person:

    def __init__(self, name, weight):
        self.name = name
        self.weight = weight

    def eat(self, weight_increased):
        self.weight += weight_increased

    def exercise(self, weight_decreased):
        self.weight -= weight_decreased

    def __str__(self):
        return f"{self.name} {self.weight}"


andy = Person('Andy', 70)
print(andy)  # Andy 70

andy.eat(2)
andy.eat(2)
print(andy)  # Andy 74

andy.exercise(3)
print(andy)  # Andy 71

      否则直接通过 print() 打印实例对象时获取的是其类和内存地址

      类私有方法名开头有两个下划线

      只能在类内部被调用的方法

      不能在类外部直接被实例对象调用访问，会报错

      class 类名:

    def __私有方法(self):
      pass

      class Demo:

    def __private_method(self):
        print('hello')

    def call_private_method(self):
        self.__private_method()


demo = Demo()

demo.__private_method()
# 报错 AttributeError: 'Demo' object has no attribute '__private_method'

demo.call_private_method()

      知道即可，不要使用

      # 类方法

      针对类自身的方法

      比如用于展示历史最高分、总数统计等

      定义时要在上一行用 @classmethod 标识

      必须有默认参数 cls 而不是 self，cls 指向当前类本身从中可获取类本身其余的类属性和类方法

      类方法通过 类名.类方法() 调用

      class 类:

    @classmethod
    def 类方法(cls):
        pass

      class Hotel:

    checkout_time = 0

    @classmethod
    def introduce(cls):
        print(cls)

    @classmethod
    def say_welcome(cls, name):
        print(f'welcome {name}')

    @classmethod
    def get_checkout_time(cls):
        print(cls.checkout_time)

    def __init__(self, customer_name):
        self.customer_name = customer_name
        Hotel.say_welcome(customer_name)
        Hotel.checkout_time += 1


Hotel.introduce()  # <class '__main__.Tool'>
Hotel.get_checkout_time()  # 0

customer_1 = Hotel('Andy')  # welcome Andy
customer_1 = Hotel('Tom')  # welcome Tom
customer_1 = Hotel('Jack')  # welcome Jack

Hotel.get_checkout_time()  # 3

      # 静态方法

      静态方法不需要访问实例属性 / 类属性，也不需要调用实例方法 / 类方法，也不需要参数

      比如仅用于输出提示信息等

      定义时要在上一行用 @staticmethod 标识

      可通过 类名.静态方法() 调用

      class 类:

    @staticmethod
    def 类方法():
        pass

      class Demo:

    @staticmethod
    def say_hello():
        print('hello~')

    def say_hello_with_name(name):
        print(f'hello {name}')

    def __init__(self, name):
        Demo.say_hello()
        Demo.say_hello_with_name(name)


Demo.say_hello()  # hello~

andy = Demo('Andy')
# hello~
# hello Andy

      # 类私有属性

      私有变量开头有两个下划线

      只能在类内部被访问使用的属性

      不能在类外部直接被实例对象直接访问和修改，会报错

      可直接定义或在 __init__ 方法中定义

      在类外部只能通过 getter 访问和 setter 修改

      class 类名:

    __私有属性 = 值

    def __init__(self):
      self.__私有属性 = 值

      class Demo:

    __private_num = 100

    def __init__(self):
        self.__private_text = 'hello'

    def get_private_num(self):
        """getter"""
        print(self.__private_num, self.__private_text)

    def set_private_num(self, num, text):
        """setter"""
        self.__private_num = num
        self.__private_text = text


demo = Demo()

print(demo.__private_num)
# 报错AttributeError: 'Demo' object has no attribute '__private_num'
print(demo.__private_text)
# AttributeError: 'Demo' object has no attribute '__private_text'

demo.get_private_num()
# 100 hello

demo.set_private_num(200, 'goodbye')

demo.get_private_num()
# 200 goodbye

      （知道即可，不要使用）

      在类外部可用 _类名__私有属性 直接访问类的私有属性

      开发中不要用，知道即可

      class Demo:

    __private_num = 100


demo = Demo()
print(demo._Demo__private_num)  # 100

      # 类属性

      仅用来记录与作为模版的当前类相关的数据

      在类内部直接定义即可

      class 类名:

    类属性 = 值

      通过 类名.类属性 访问和修改

      可在类外部被直接访问到

      class Tool:

    time_used = 0

    def __init__(self, name):
        self.name = name

    def add(self):
        Tool.time_used += 1


print(Tool.time_used)  # 0

tool_1 = Tool('斧子')
tool_1.add()
tool_1.add()

tool_2 = Tool('锤子')
tool_2.add()

print(Tool.time_used)  # 3

      从实例上获取类属性（千万不要用）

      • 根据 Python 的属性向上查找机制，通过 实例对象.类属性 也可以获取类属性，但若实例上存在同名属性时则会优先获取实例的
      • 通过 实例对象.类属性 = 新值 修改类属性时并不会影响到类属性，若实例对象上有该同名属性则只会修改实例自身的该属性的值，若没有则只会直接给实例上追加一个值为新值的属性
        综上，不要用这种做法获取和修改类属性
      class Tool:

    time_used = 0

    def __init__(self, name):
        self.name = name
        self.time_used = 0

    def add(self):
        Tool.time_used += 1
        self.time_used += 1


tool_1 = Tool('斧子')
tool_1.add()
tool_1.add()

tool_2 = Tool('锤子')
tool_2.add()

print(Tool.time_used)  # 3
print(tool_1.time_used)  # 2
print(tool_2.time_used)  # 1

      父类 == 基类
      继承 == 派生

      子类会继承父类中的所有属性和方法

      Python3 中不子类指定父类时默认继承 object 基类中的内置属性与方法

      逗号多个父类

      # 方法重写

      同名覆盖，自身优先

      子类和父类中尽量不要有重名方法以免子类的重新覆盖

      class Father:

    def introduce(self):
        print("it's father")


class Son(Father):

    def introduce(self):
        print("it's son")

    def say(self):
        self.introduce()


demo = Son()

demo.say()  # it's son

      若想在子类中使用父类的方法需要使用 super() 调用父类的方法

      详见下文 super()

      class Father:

    def introduce(self):
        print("it's father")


class Son(Father):

    def introduce(self):
        print("it's son")

    def say(self):
        super().introduce()


demo = Son()

demo.say()  # it's father

      # super()

      super() 用于调用父类中的某一个方法

      原理是 super() 会把子类的对象转换为父类的对象

      Python 3 的 super() 写法：

      class 子类(父类):

   def __init__(self):
      super().__init__(参数)

   def 实例方法(self):
    	super().父类中方法(参数)

      Python 2 的 super() 写法：

      class 子类(父类):

   def __init__(self):
      super(父类名, self).__init__(参数)

   def 实例方法(self):
    	super(父类名, self).父类中方法(参数)

      # 多态

      调用相同的父类方法，但传入不同对象参数执行生成不同结果

      class Animal:

    def __init__(self, name):
        self.name = name

    def laugh(self):
        print(f'{self.name} is laughing')


class Dog(Animal):
    pass


class Cat(Animal):
    pass


class Person:

    def __init__(self, name):
        self.name = name

    def play_with_pet(self, pet):
        print(f'{self.name} is playing with {pet.name}')
        pet.laugh()


andy = Person('Andy')
dog = Dog('doggy')
cat = Cat('catty')

andy.play_with_pet(dog)
# Andy is playing with doggy
# doggy is laughing

andy.play_with_pet(cat)
# Andy is playing with catty
# catty is laughing

      # 单例设计模式

      单例设计模式是指是类只创建仅唯一一个实例对象

      比如：音乐播放对象一次只能放一首歌，只有一个回收站

      创建实例对象时 内置方法 __new__ 会被默认调用为实例对象分配内存空间，此时只要永远返回一个固定的引用地址，就可实现无论调用执行多少次 类名() 都只会创建内存地址相同的实例对象

      class MusicPlayer:

    instance = None

    def __new__(cls):
        if cls.instance is None:
            cls.instance = super().__new__(cls)
        return cls.instance


a = MusicPlayer()
print(id(a))
# 4431674912

b = MusicPlayer()
print(id(b))
# 4431674912

print(id(a) == id(b))  # True
