      # Less

      Less 基础

      Vue-Cli + Less

      vue-cli 生成项目时勾选 CSS Pre-processors (CSS 预处理器)

      1. 组件中使用

      <style lang="less" scoped></style>

<!-- 删除scoped 全局范围内生效 -->
<style lang="less"></style>

      2. 组件中/Scss 文件中导入

      @import "../src/static/common.less";

      3. main.js 中导入

      import "./static/common.less";

      若生成项目时没配置，需要安装 less 和 less-loader

      但直接安装less-loader会报错，因为Vue-Cli4采用的是webpack4，而直接安装的less-loader的最新版本是面向webpack5+，所以需要给less-loader降级，建议版本7

      npm i less-loader@7.0.0

      因为修改了配置文件，需要重起脚手架

      # Sass

      Sass/Scss 基础

      Vue-Cli

      vue-cli 生成项目时勾选 CSS Pre-processors (CSS 预处理器)

      1. 组件中使用

      <style lang="scss" scoped></style>

<!-- 删除scoped 全局范围内生效 -->
<style lang="scss"></style>

      2. 组件中/Scss 文件中导入

      @import "../src/static/common.scss";

      3. main.js 中导入

      import "./static/common.scss";

      Create-React-App

      Cannot find module 'sass'

      React 提前配置了 loader，但是没有配置 Sass 模块，需要手动下载 Sass

      1. 下载
      npm i sass
# or
yarn add sass
      1. 因为修改了配置文件，需要重起脚手架

      Module build failed: TypeError: this.getResolve is not a function

      sass 和 sass-loader 的版本过高导致的编译错误，降低版本即可

      npm uninstall sass-loader（卸载当前版本）
npm install sass@5.1.1 sass-loader@10.0.1
