# 解析器

TS目前无法被浏览器解析，需要先全局安装typescript解析器

通过解析器将TS文件解析为JS文件

常用如下：

  • tsc
  • ts-node

# tsc

全局安装typescript解析器

npm i -g typescript

tsc -V
tsc --version

通过tsc命令将文件解析为同名JS文件

tsc 文件.ts

解析后同路径下生成同名.js文件

TS 中的let、const 在编译为JS后都转换为 var

|- 文件.ts
|- 文件.js  # 解析后生成的文件

然后通过node命令运行解析好了的 JS 文件

node 文件.js

# ts-node

可全局安装ts-node

使.ts文件不通过解析就可在Node.js环境中直接运行

npm i -g ts-node

ts-node --version

通过ts-node命令直接运行 JS 文件

ts-node 文件名.ts
